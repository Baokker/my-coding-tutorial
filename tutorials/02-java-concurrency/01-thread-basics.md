## 1. еҲӣе»әзәҝзЁӢзҡ„дёүз§Қж–№ејҸ

```java
// ж–№ејҸдёҖпјҡз»§жүҝ Thread зұ»пјҲдёҚжҺЁиҚҗпјҢJava еҚ•з»§жүҝпјү
class MyThread extends Thread {
    @Override
    public void run() {
        System.out.println(Thread.currentThread().getName() + ": иҝҗиЎҢдёӯ");
    }
}

// ж–№ејҸдәҢпјҡе®һзҺ° Runnable жҺҘеҸЈпјҲжҺЁиҚҗпјү
class MyTask implements Runnable {
    @Override
    public void run() {
        System.out.println(Thread.currentThread().getName() + ": иҝҗиЎҢдёӯ");
    }
}

// ж–№ејҸдёүпјҡе®һзҺ° Callable жҺҘеҸЈпјҲеҸҜд»Ҙиҝ”еӣһз»“жһң + жҠӣејӮеёёпјү
class MyCallable implements Callable<String> {
    @Override
    public String call() throws Exception {
        Thread.sleep(1000);
        return "д»»еҠЎе®ҢжҲҗ";
    }
}

public class ThreadDemo {
    public static void main(String[] args) throws Exception {
        // ж–№ејҸдёҖ
        new MyThread().start();

        // ж–№ејҸдәҢ
        new Thread(new MyTask()).start();

        // ж–№ејҸдәҢпјҲLambda з®ҖеҶҷпјҢжңҖеёёз”Ёпјү
        new Thread(() -> {
            System.out.println("Lambda зәҝзЁӢ: " + Thread.currentThread().getName());
        }).start();

        // ж–№ејҸдёү
        FutureTask<String> future = new FutureTask<>(new MyCallable());
        new Thread(future).start();
        String result = future.get(); // йҳ»еЎһзӯүеҫ…з»“жһң
        System.out.println("Callable з»“жһң: " + result);
    }
}
```

## 3. synchronized е…ій”®еӯ—

```java
public class SynchronizedDemo {

    private int count = 0;

    // ж–№ејҸдёҖпјҡеҗҢжӯҘж–№жі•пјҲй”Ғ = this еҜ№иұЎпјү
    public synchronized void increment() {
        count++;
    }

    // ж–№ејҸдәҢпјҡеҗҢжӯҘеқ—пјҲжӣҙзҒөжҙ»пјҢеҸҜд»ҘжҢҮе®ҡй”ҒеҜ№иұЎпјү
    private final Object lock = new Object();
    public void incrementWithBlock() {
        synchronized (lock) {
            count++;
        }
    }

    // ж–№ејҸдёүпјҡйқҷжҖҒеҗҢжӯҘж–№жі•пјҲй”Ғ = Class еҜ№иұЎпјү
    private static int globalCount = 0;
    public static synchronized void staticIncrement() {
        globalCount++;
    }

    public static void main(String[] args) throws InterruptedException {
        SynchronizedDemo demo = new SynchronizedDemo();

        // еҗҜеҠЁ 100 дёӘзәҝзЁӢеҗҢж—¶йҖ’еўһ
        Thread[] threads = new Thread[100];
        for (int i = 0; i < 100; i++) {
            threads[i] = new Thread(() -> {
                for (int j = 0; j < 1000; j++) {
                    demo.increment();
                }
            });
            threads[i].start();
        }

        // зӯүеҫ…жүҖжңүзәҝзЁӢе®ҢжҲҗ
        for (Thread t : threads) {
            t.join();
        }

        System.out.println("жңҖз»Ҳи®Ўж•°: " + demo.count);  // 100000
    }
}
```

## 4. wait / notify жңәеҲ¶

```java
public class ProducerConsumer {
    private final Queue<Integer> queue = new LinkedList<>();
    private final int capacity = 5;

    // з”ҹдә§иҖ…
    public synchronized void produce(int item) throws InterruptedException {
        while (queue.size() == capacity) {
            wait();  // йҳҹеҲ—ж»ЎдәҶпјҢзӯүеҫ…ж¶Ҳиҙ№иҖ…ж¶Ҳиҙ№
        }
        queue.add(item);
        System.out.println("з”ҹдә§: " + item + " йҳҹеҲ—еӨ§е°Ҹ: " + queue.size());
        notifyAll();  // йҖҡзҹҘж¶Ҳиҙ№иҖ…
    }

    // ж¶Ҳиҙ№иҖ…
    public synchronized int consume() throws InterruptedException {
        while (queue.isEmpty()) {
            wait();  // йҳҹеҲ—з©әдәҶпјҢзӯүеҫ…з”ҹдә§иҖ…з”ҹдә§
        }
        int item = queue.poll();
        System.out.println("ж¶Ҳиҙ№: " + item + " йҳҹеҲ—еӨ§е°Ҹ: " + queue.size());
        notifyAll();  // йҖҡзҹҘз”ҹдә§иҖ…
        return item;
    }

    public static void main(String[] args) {
        ProducerConsumer pc = new ProducerConsumer();

        // з”ҹдә§иҖ…зәҝзЁӢ
        new Thread(() -> {
            for (int i = 0; i < 10; i++) {
                try { pc.produce(i); } catch (InterruptedException e) { break; }
            }
        }).start();

        // ж¶Ҳиҙ№иҖ…зәҝзЁӢ
        new Thread(() -> {
            for (int i = 0; i < 10; i++) {
                try { pc.consume(); } catch (InterruptedException e) { break; }
            }
        }).start();
    }
}
```
